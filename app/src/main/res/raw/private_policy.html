Android Permissions

This section describes briefly why specific Android permissions are required.

  • Camera

    The camera permission is required to read QR codes, which contains information about Peer ID's or data content which should be stored on the users device. The user is responsible what kind of data it stores on it's device.

  • Read External Storage

    The external read storage permission is required to read files locally and add them to the local IPFS node.

  • Write External Storage

    The external storage write permission is required to write files from local IPFS node to the the general "Download" directory.

  • Foreground Service

    The foreground service permission is required to run the IPFS node over a longer period of time.

  • Internet

    Kind of precondition for running the local IPFS node.

  • Access Network State

    The access to the network state is required, so that the best configuration for the local IPFS node can be configured.
